# Scan2Learn Project

Scan2Learn is a full-stack intelligent learning assistant that extracts text from handwritten notes and PDFs in Indian regional languages using OCR and NLP techniques.

It transforms raw notes into structured, personalized learning experiences — including summaries, grammar corrections, quizzes, weak-area detection, and adaptive practice schedules.

## 🚀 Features
1. OCR-Based Text Extraction

Extracts text from:
Handwritten notes, printed PDFs, supports Indian regional languages, built using Tesseract OCR and OpenCV.

2. Smart Summarization

Generates concise summaries of long content. Transformer-based NLP models via HuggingFace.

3. Grammar Correction

Detects and corrects grammatical errors. Improves readability and clarity.

4. Transliteration

Converts regional language text into different scripts. Powered by IndicNLP.

5. Auto-Generated Quizzes

Generates practice questions automatically. Helps reinforce learning concepts.

6. Weak-Area Analysis

Identifies topics where students struggle. Tracks performance trends.

7. Personalized Practice Schedules

Suggests custom study plans. Adaptive learning workflow.

## 🏗️ Tech Stack
React.js, HTML, CSS, JavaScript, Python, Flask, NLP & ML, HuggingFace Transformers, IndicNLP, NLTK, OCR & Image Processing, Tesseract OCR, OpenCV, Version Control, Git & GitHub


⚙️ How It Works

- User uploads handwritten notes or PDFs.
- OCR extracts raw text.
- NLP pipeline processes the text:
- Cleaning
- Summarization
- Grammar correction
- Quiz generation
- Performance tracking module analyzes results.
- System generates personalized study recommendations.
